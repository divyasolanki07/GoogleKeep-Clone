# GoogleKeep Clone

> A modern note-taking web app inspired by Google Keep — built with **React** and **Vite**.


## 🔥 Project Highlights

- **Simple note creation** with instant cards
- **Clean UI** with icon-driven actions and a polished layout
- **Modal view** for note details
- **Responsive design** for desktop and mobile
- **Fast user experience** powered by Vite


## 🌟 What’s Included

- 📌 Add new notes quickly
- 🗑️ Delete notes with a single click
- 📝 View note details inside a modal popup
- 🎨 Attractive card-based layout
- 🌍 Mobile-friendly interface


## 🧠 Why This App Works

This project is a great starter app for learning React component structure and modern frontend tooling. It uses a clear component hierarchy and keeps the UI simple for daily note-taking.


## 🧩 Folder Structure

- `src/`
  - `components/`
    - `CreateNote/` — note creation form
    - `Header/` — app header
    - `NoteCard/` — display note cards
    - `NoteModal/` — detail modal view
    - `Sidebar/` — sidebar navigation
  - `utils/` — storage helpers and support functions
- `public/` — static public assets
- `index.html` — app HTML entry point
- `package.json` — dependencies and scripts


## ⚙️ Run the Project

Use these commands from the project folder:

- `npm install` — install dependencies
- `npm run dev` — start development server
- `npm run build` — build production files
- `npm run preview` — preview production build
- `npm run lint` — run ESLint checks


## 📌 Usage Guide

1. Start the app with `npm run dev`
2. Use the note form to create a new note
3. Notes appear as cards on the screen
4. Click a card to open the note modal
5. Edit or delete notes as needed
